# FLTERS - Termux 配置

**FLTERS** 是一个专为 Termux 打造的配置，旨在快速搭建美观、高效的终端开发环境。

## ✨ 特性

- **美观界面**：采用 Catppuccin 主题，搭配真彩色和光标拖尾特效
- **高效工具链**：集成 LazyVim、Yazi、Starship/Oh My Posh、fzf、ripgrep 等热门工具
- **开箱即用**：预置常用别名、Zsh 插件和 Neovim 快捷键

## 📦 包含组件

| 组件 | 说明 |
|------|------|
| Zsh + Oh My Zsh | 强大的 Shell，支持丰富插件 |
| Powerlevel10k / Starship | 炫酷的命令行提示符 |
| LazyVim | 现代化 Neovim 配置，懒加载机制 |
| Catppuccin 主题 | 柔和护眼的配色方案 |
| Yazi | 终端文件管理器，支持图片/视频预览 |
| fzf + ripgrep | 模糊搜索和全文检索工具 |
| zsh-autocomplete | 智能命令补全 |
| fast-syntax-highlighting | 实时语法高亮 |

❤️FLTERS - 让 Termux 更优雅❤️
